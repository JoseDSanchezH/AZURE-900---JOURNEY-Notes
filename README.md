Dedicated Server- One Physical Machine dedicated to a single business. Runs a single web-app/site. (Very Expensive, high-maintenance, High Security)

Virtual Private Server (VPS) - one physical machine dedicated to a single business. The physical machine is virtualized into sub-machines. So a machine within a machine. (Better Utilization and isolation of resources. 

Shared Hosting - one physical machine shared by hundreds of businesses. This relies on most tenants underutilizing their resources. (Very cheap, Limited functionality, poor isolation). 

Cloud hosting - Multiple physical machines that act as one system. (Flexible, Scalable, Secure, Cost-effective, High configurability). 


# Common Cloud services

1- Compute: A virtual computer that can run applications, programs, and code. 
2- Storage: A virtual hard drive that can store files. 
3- Networking: A virtual network can define internet connections or network isolations. 
4- Databases: A virtual database can store and report data, or a database for a general-purpose web application. 

## Services 

SaaS - Software as a Service - A product that is run and managed by the service provider (Salesforce, GMAIL, OFFICE 365). **Customers**

PaaS - Platform as a Service - Focus on the deployment and management of your apps (examples are Heroku, Google App Engine). **Developers**

IaaS - Infrastructure as a Service- Basic building blocks for cloud IT. Access to networking features, computers, and data storage space. (AZURE, AWS, ORACLE) **Admins**


## Azure Deployment Models

Public cloud - Everything is built on the Cloud provider. AKA Cloud- Native

Private Cloud - Everything is built on the company's datacenters. AKA On- Premise
Cloud can be OpenStack

Hybrid - Using both On-premise and cloud service providers.
